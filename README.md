## **Project Structure**
```
  /frontend
      - frontend VUE application
  /backend
      - backend SpringBoot framework
  /database
      - database creation and data insertion script
  /documents
      - documents for system API 
```

Front-end:  
Install node modules:  
**npm install**  
Run the program   
**npm run serve**  
Build program to production  
**npm run build**
